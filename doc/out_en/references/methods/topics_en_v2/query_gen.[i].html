query[i]

NAME

[i] - Access the result set by index.

SYNOPSIS

query[i]

CATEGORY

Query

DESCRIPTION

Access the query result set by index.

Examples

Get a result set and access it by index.

var query = db.foo.bar.find({age:{$gt:10}})
println( query[0] )
