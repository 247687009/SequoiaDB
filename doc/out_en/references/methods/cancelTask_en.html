db.cancelTask()

db.cancelTask(<id>,[isAsync]) Cancel the task

Parameter Description

Parameter name Parameter type Description Are Required
id Integer Task ID Yes
isAsync Boolean Whether asynchronous No

Example

  • Stop splitting tasks.

    var taskid1 = db.test.test.splitAsync("db1", "db2", 50);
db.cancelTask( taskid1, true )
