Monitoring Management

PHP monitoring supports IE 9+, Chrome, Firefox and other popular browsers.

  • 1. Start PHP service
    1.1 On the host that installed SequoiaDB, execute bin/sdbwsart under sequoiadb path. For example:
    /opt/sequoiadb/bin/sdbwsart

    1.2 1.2 Input the host address and port number in the browser of last step. http://192.168.1.215:8080/

  • 2. Connect monitoring page

    2.1 In Connection Management, insert the address in <连接地址> and click <连接测试>. After test succeed click <连接>.

  • 3. 3. Create collectionspace, collection and insert data.

    3.1 To create collectionspace, click button in the <工具栏>. Then, you may input the <集合空间名> and click <确定>.

    3.2 To create collection in a CS, choose <属性> of the CS(like CS foo here), then click the button in the <工具栏>. Input <集合名> and click <确定>.

    3.3 To insert data, choose the collection (like bar here), click the <数据> of the collection. Use the button in the <工具栏>, input the JSON record, click <确定>.

    3.4 Formatting the JSON record, use the button in the <工具栏>.

  • 4. Cluster Management

    4.1 Check the cluster, click <集群管理> in the navigation bar. Then you can manage the clusters.
