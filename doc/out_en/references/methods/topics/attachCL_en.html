db.collectionspace.collection.attachCL()

db.collectionspace.collection.attachCL(<subCLFullName>, <options>)

Colletion of child partition mounted under the main partition collection.

Parameter Description

Parameter name Parameter type Description Are Required
subCLFullName string The partition set name(the name that contains a collection of space) Yes
options json object The boundaries, contains two fields "LowBound"(the value of the left section) and "UpBound"(R interval value), for example: {LowBound:{a:0},UpBound:{a:100}, means to take the field "a" range interval: [0, 100) Yes

Example

  • mount the specified range in the primary partition of the sub-set of partitions collection.

    db.foo.year.attachCL("foo2.January",{LowBound:{date:"20130101"},UpBound:{date:"20130131"}}
