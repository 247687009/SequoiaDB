Python 开发环境搭建

获取驱动开发包

从 http://www.sequoiadb.com 下载对应操作系统版本的 SequoiaDB 驱动开发包。

配置开发环境

  • Linux

    1. 解压下来的驱动开发包；得到 python 目录。

    2. 将 python 目录下的 bson 和 pysequoiadb 目录拷贝到开发工程目录中（建议放在第三方库目录下）。

  • Windows

    暂未推出 Windows 驱动开发包。
