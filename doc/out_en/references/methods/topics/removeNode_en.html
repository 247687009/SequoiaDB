rg.removeNode()

rg.removeNode(<host>,<service>,[config])

Delete slice group specified node.

Parameter Description

Parameter name Parameter type Description Required
host string node hostname Yes
service string node port number Yes
config json node configuration information. No

Format

Definition Format rg.removeNode() method has three parameters: host, service, config, as shown above, the following format: 
("<hostname>","<port number>"[,{<configParam>:value,...}])

Example

  • Delete node command follows slice group shard 
    rg.removeNode("vmsvr2-suse-x64",11800)
Note:

Delete the specified node must exist, otherwise abnormal.
