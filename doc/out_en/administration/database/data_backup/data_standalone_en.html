Standalone Mode

In standalone node,users must completely stop the database before archiving all the files in database and index direcroties.

In the case of specifying configuration files,the database and index directories are the specified by the dbpath and indexpath in the configuration file.By default,the value of them are same.

And if configuration file doesn't exist,database directory is specified at the time of starting database.
