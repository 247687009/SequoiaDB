Python API

History Updates:

Version 1.10

1. New interfaces class Lob:

close, close the Lob object that created before, to refresh the data
readto, read data from Lob objects
write, insert data into Lob
seek, skip to a specified position of data
get_oid, get the oid of a Lob
get_size, get the size of Lob object
get_create_time, get the creating time of a LOB

2. New interfaces of Collection:

create_lob, create a Lob object in current collection
remove_lob, remove a Lob object in current collection
get_lob, get Lob object with specific oid in current collection
list_lobs, list all Lobs in current collection

For details, please see the relevant API.
