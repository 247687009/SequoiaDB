sql drop index

drop index 语句

用于删除集合中的索引。

语法

drop index <index_name> on <cs_name>.<cl_name>

<index_name>：索引名

<cs_name>：集合空间名

<cl_name>：集合名

示例

  • 删除集合空间 foo 中集合 bar 下名为 test_index 的索引名

    db.execUpdate("drop index test_index on foo.bar") //等价于 db.foo.bar.dropIndex("test_index")
