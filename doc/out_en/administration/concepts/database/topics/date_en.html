Date

Concept

In SequoiaDB, date is in the format of YYYY-MM-DD. It is transformed into a 4-byte integer when it is stored in the database.

Format

The format of date is:

{ "$date" : "<YYYY-MM-DD>" }

Sample

For example: 

{ "createTime" : { "$date" : "2012-05-12" } }
